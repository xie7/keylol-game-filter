# 其乐 Keylol 游戏折扣筛选器

![预览图](https://img.shields.io/badge/版本-1.0.0-blue)
![油猴脚本](https://img.shields.io/badge/油猴脚本-✓-green)

## 功能特点

- 🎮 智能筛选：支持价格范围、游戏评价、语言支持筛选
- 🌙 日夜模式：一键切换深色/浅色主题
- 🖱️ 拖拽调整：可调节悬浮窗大小，智能限制不遮挡表格
- 🎨 美观界面：渐变背景、毛玻璃效果、增强阴影
- ⚡ 即时反馈：筛选结果实时高亮显示

## 安装方法

1. 安装油猴脚本管理器（Tampermonkey 或 Violentmonkey）
2. 点击安装链接：[安装脚本](https://raw.githubusercontent.com/xie7/keylol-game-filter/main/keylol-game-filter.user.js)
3. 访问其乐 Keylol 论坛的游戏折扣帖子，左下角会出现筛选面板

## 使用方法

1. 打开其乐 Keylol 的游戏折扣帖子（如：https://keylol.com/forum.php?mod=viewthread&tid=1034188）
2. 页面左下角会出现游戏筛选器悬浮窗
3. 选择筛选条件：
   - 💰 价格范围：0~20、21~50、51以上
   - ⭐ 游戏评价：全部、好评如潮、特别好评、多半好评、褒贬不一
   - 🌐 语言支持：简体中文、繁体中文
4. 点击"确定筛选"应用条件
5. 点击右上角🌙/☀️切换日夜模式
6. 拖动右下角三角区域调整窗口大小

## 技术特性

- 仅筛选当前页面数据，不跨页获取
- 智能尺寸限制：最小280x350px，最大500x600px
- 自动检测表格位置，防止窗口覆盖数据
- 支持页面动态加载（翻页时自动重新初始化）
